# Fonts

Drop your FH Oscar font files here.

Expected filenames (Regular is required, others are optional):

- `FHOscar-Regular.otf`
- `FHOscar-Medium.otf`
- `FHOscar-Bold.otf`

If you only have one file, rename it to `FHOscar-Regular.otf`. Other weights
will fall back gracefully.

`.woff2` versions of the same files are also loaded if present (preferred by
browsers for performance). Just drop them in with the same base name, e.g.
`FHOscar-Regular.woff2`.
